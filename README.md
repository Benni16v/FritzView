## Features

- Modern Dashboard
- Modular Plugin System
- Theme Engine
- LCD4Linux Backend
- TR-064 Support
- Call Monitor
- Phonebook
- Contact Pictures
- DSL Statistics
- Online History
- CRC History
- CPU / RAM / Temperature
- Multi Page Display
- Retro Themes
