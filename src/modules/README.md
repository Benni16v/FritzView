Deutsch
-------

Dieses Verzeichnis enthält alle Datenquellen.

Jedes Modul fragt genau einen Bereich der FRITZ!Box ab.

Beispiele:

wan.sh
DSL
WLAN
Telefonie
Mesh
Hosts

Neue Module können einfach ergänzt werden.

-------------------------------

English
-------

This directory contains all FRITZ!Box data providers.

Each module is responsible for exactly one subsystem.

Examples:

WAN
DSL
WiFi
Phone
Mesh
Hosts

Additional modules can easily be added.
