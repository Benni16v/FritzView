Deutsch
-------

Hardwaretreiber.

Jede Datei unterstützt genau ein Display.

Beispiele:

console.sh
lcd4linux.sh
pearl.sh
samsung.sh

Neue Displays können ergänzt werden, ohne den Rest des
Programms zu ändern.

-------------------------------

English
-------

Hardware display drivers.

Each driver supports one display device.

Examples:

console
Pearl
Samsung
LCD4Linux

New drivers can be added without modifying the rest of
FRITZ!View.
